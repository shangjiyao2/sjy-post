

## 分时电价执行异常监测列表查询

### 1. 接口信息

- **接口路径**: `/api/yxjcscene/v1/yxjcSceneTouPriceException/list`
- **请求方式**: POST
- **Content-Type**: application/json
- **接口描述**: 分页查询分时电价执行异常监测列表，支持按监测日期、区县、客户编号、客户名称筛选

### 2. 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| monitorDate | String | 否 | 监测日期（格式：yyyy-MM-dd） |
| countyUnit | String | 否 | 区县（县单位） |
| customerCode | String | 否 | 客户编号（模糊查询） |
| customerName | String | 否 | 客户名称（模糊查询） |
| pageNum | Integer | 否 | 页码，默认1 |
| pageSize | Integer | 否 | 每页条数，默认10 |

### 3. 请求示例

```json
{
  "monitorDate": "2026-02-01",
  "countyUnit": "城东县",
  "customerCode": "KH202601",
  "customerName": "华兴",
  "pageNum": 1,
  "pageSize": 10
}
```

### 4. 响应参数



| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | Long | 主键ID |
| monitorDate | String | 监测日期 |
| countyUnit | String | 县单位 |
| belongUnit | String | 所属单位 |
| customerCode | String | 客户编号 |
| customerName | String | 客户名称 |
| energyAddress | String | 用能地址 |
| customerCategory | String | 客户分类 |
| energyCategory | String | 用能类别 |
| voltageLevel | String | 承压等级 |
| industryCategory | String | 行业分类 |
| contractCapacity | BigDecimal | 合同容量(kVA) |
| operatingCapacity | BigDecimal | 运行容量(kVA) |
| settlementPower | BigDecimal | 结算电量(kWh) |
| valleyPower | BigDecimal | 谷段电量(kWh) |
| valleyPowerRatio | BigDecimal | 谷段电量占比(%) |
